# GO Fest 2026 グローバル
[GoFest 2026 東京][tokyo] も行きましたが、グローバルは近所で参加しました。ただ、自分は遭遇しなかったのですが、子どもたちがレイドに参加できない不具合に遭遇してしまいました。そうなると、こちらもトラブル対応で時間を取られてしまったりします。不具合を完全になくすのは難しいですが、あまり大きいものは避けてもらいたいものです。

そんな Go Fest 2026 グローバルだけでなく、前段の大きなる軌跡のところからまとめての結果を振り返ります。

# 結果
## いろちがい
{% include thumb.html path="2026/07/ss-20260712-01.jpg" alt="いろちがい" %}
{% include thumb.html path="2026/07/ss-20260712-02.jpg" alt="いろちがい" %}

結構持っているいろちがいが多かった印象です。エアームドは土曜日のメガミュウツーXの合間に捕まえました。ミュウツーの色違いはメガミュウツーXが1匹のみ。Y は色違いでませんでした。。。

オリジンパルキアは色違いを持っていなかったので、狙っていましたがあっさりと一戦目で出てきて逆に拍子抜け。あくうせつだんを持っていなかったので、すごいわざマシンスペシャルで覚えさせました。

## でんせつ＆まぼろし
{% include thumb.html path="2026/07/ss-20260712-03.jpg" alt="でんせつ" %}
{% include thumb.html path="2026/07/ss-20260712-04.jpg" alt="でんせつ" %}
{% include thumb.html path="2026/07/ss-20260712-05.jpg" alt="でんせつ" %}

これまでのほぼすべてのレイドボスが復刻って、やりすぎですよね。とてもじゃないけど狙ってできる数ではなかったと感じました。とはいえ、登場する中で持っていなかった、ノーマルフォルムのデオキシスをゲットできたのは幸運だったと思います。

キュレムはホワイトキュレムの色違いを作りたかったので少し回数多めにやりました。ミュウツーは無料のレイドパスが 9 枚ずつあったのでそれなりの回数できましたが、色違いは 1 体のみでした。

あ、ルギアは Go パス 6月分だと思います。
