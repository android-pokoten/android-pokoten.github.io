# GO Fest 2026 東京
リアルイベントは初参加です。まち探検チケットしか購入できなかったのですが、お台場にも行って雰囲気を感じてきました。本当にお祭りという感じで、普段のゲームプレイとは違った楽しさがありました。5/31 だったので、午前中は曇り空で多少暑さも和らいだ時間帯だったのもよかったかもしれません。

公園探索チケットとは違い、メガミュウツーのユニティレイドやゼラオラは入手できませんが、いろいろな特典があったのでその結果を振り返ります。

# 結果
## いろちがい

{% include thumb.html path="2026/06/ss-20260602-01.jpg" alt="いろちがい" %}

いずれも初入手。アンノーンは 2 匹捕まえられました。悪くはないのですが、レイド産や特別コスチューム系の色違いが無しなのは少し残念です。また、イベント期間中にアローラロコンが野生沸きしていました。ロコンのコミュニティディに参加できなかったので、色違いが欲しかったのですが結局出なかったのは残念でした。

## でんせつ

{% include thumb.html path="2026/06/ss-20260602-02.jpg" alt="でんせつ" %}

無料のレイドパスが 9 枚ありました。(前日持ち越し分の 1 枚もあったので、パルデアケンタロスもやりました)

特に意識していなかったのですが、スイクン、カイオーガ、ミュウツーが 3 回ずつでした。いずれも色違いもすでに持ってはいましたが、今回色違いは無し。ロケーション背景が付いたのも 2 体のみ。まあ、無料分だけだとこんなものですかねぇ

## ロケーション背景

{% include thumb.html path="2026/06/ss-20260602-03.jpg" alt="ロケーション背景" %}

背景が付いたのは 2 匹のみ。タイムチャレンジ 4 種をコンプリートし、ロケーション背景付きのラプラスも入手したのでトータルでは 3 匹でした。

## 特別コスチューム

{% include thumb.html path="2026/06/ss-20260602-04.jpg" alt="特別コスチューム1" %}
{% include thumb.html path="2026/06/ss-20260602-05.jpg" alt="特別コスチューム2" %}

素のゼニガメも混じってますが、コスチュームの色違いも出ませんでした。

## 特別なわざ

忘れてましたが、「だいちのちから」を覚えたガブリアスと、「ハードプラント」を覚えたジュナイパーも手に入れました。ただ、一緒にまわっていた子どもたちは色違いのバンギラスを手に入れていたりしていて、うらやましかったです。

## アンノーン

{% include thumb.html path="2026/06/ss-20260602-06.jpg" alt="アンノーン" %}

おこうに反応するのと、毎時 58 分から 2 分間、アンノーンが大量沸きしていました。たった 2 分しかないので、自分の狙うアンノーンがいるかどうかをゆっくり探す余裕もないですが、あと 5 種類・・・。イベント前の状態を控えていませんでしたが、コンプリートは遠いようです。

## メガミュウツー
メガミュウツーは X を選びました。こちらも色違いにはなりませんでした。子どもたちは色違いだったそうです。「タスク産は必ず色違いだと思ったからそんなに驚かなかった」なんて言ってました。ちゃんと通常のも出るんだよ・・・
